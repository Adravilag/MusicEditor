# MusicEditor

Este proyecto es un pequeño editor de música estilo pixel‑art inspirado en juegos como **Undertale**. Utiliza [pygame](https://www.pygame.org/) para mostrar una cuadrícula de notas tipo secuenciador. Haz clic en las celdas para activar o desactivar notas y presiona la barra espaciadora para reproducir la secuencia.

## Requisitos

- Python 3
- pygame

## Ejecución

```bash
python music_editor.py
```

Presiona **Espacio** para escuchar la pista y cierra la ventana para salir.
